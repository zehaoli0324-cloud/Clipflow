# NVIDIA GTC Taipei 2026 — 旁白 v3（复利投研社风格）

## 风格定位
- 第一人称 "我看了演讲，终于明白..."
- 三个关键词架构：Agent / AI工厂 / 个人AI电脑
- 投资分析视角，讲"未来十年谁最赚钱"
- 短句+冲击力+每段有观点输出
- 口语化，像在跟朋友聊天分享发现

时长：~3分54秒 / ~950字

## 核心变化

| 维度 | v2 | v3 |
|------|-----|-----|
| 结构 | 按时间线叙事 | 三个关键词架构（Agent/AI工厂/个人AI） |
| 开场 | 描述事件 | "我看了，终于明白"——认知悬念 |
| 语言 | 设问+结论 | 口语化分享，像朋友聊天 |
| 收尾 | 客观陈述 | 冲击力"你在牌桌上" + 免责声明 |
| 节奏 | 匀速 | 短句+停顿+数据冲击 |

## 建议
- 语速：自然聊天感，不要太快
- 每段动画卡之后停顿一下，给观众消化的时间
- 最后一句"个人观点，仅供参考"要说得轻一点、真诚一点
- 背景音乐选有科技感但不抢话的电子配乐
